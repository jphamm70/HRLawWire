# HRLawBrief

Daily curated employment law, HR compliance, and workplace policy news for HR professionals and employment attorneys.

## About

HRLawBrief is a Drudge Report-style news aggregator focused exclusively on HR and employment law topics. Updated daily with the most important headlines across:

- EEOC & Discrimination
- NLRB & Labor Relations
- Wage & Hour
- ADA & Accommodations
- Leave & Benefits
- DEI & Workplace Culture
- AI & HR Tech
- Workplace Safety
- Immigration
- Federal Policy
- State & Local Updates

## How It Works

The site loads headlines from `headlines.json`, making daily updates simple:
1. Update the JSON file with today's headlines
2. Commit and push
3. Site automatically updates via Netlify

## File Structure

```
/
├── index.html      # Main website
├── headlines.json  # Daily headlines data
└── README.md       # This file
```

## Deployment

Hosted on Netlify with automatic deploys from this repository.
